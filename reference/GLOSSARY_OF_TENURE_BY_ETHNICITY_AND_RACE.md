# Number of households by owning/renting by Hispanic/not by OMB race

Number of households by owning/renting by Hispanic/not by OMB race

## Usage

``` r
GLOSSARY_OF_TENURE_BY_ETHNICITY_AND_RACE
```

## Format

description An object of class `tbl_df`/`tbl`/`data.frame` with 93 rows
and 7 columns

- Vintage:

  `<int>` which decennial census

- Group:

  `<chr>` the table in the decennial census product

- Index:

  `<int>` the row in the table

- Variable:

  `<chr>` the full munged name of the variable

- Tenure:

  `<chr>` whether the households are owners or renters

- Ethnicity:

  `<chr>` whether the householders are Hispanic or not

- Race:

  `<chr>` which OMB race the householders identify as
