# Variables from Demographic and Housing Characteristics survey.

Variables from Demographic and Housing Characteristics survey.

## Usage

``` r
DHC_VARIABLES
```

## Format

a data frame with five columns

- Concept:

  `<chr>` a verbal description of the group's contents

- Group:

  `<chr>` the code that designates the variable's group

- Index:

  `<int>` the row number of the variable

- Variable:

  `<chr>` the complete variable name

- Details:

  `<lst>` the details for this variable

## Source

<https://api.census.gov/data/2020/dec/dhc/variables.html>
